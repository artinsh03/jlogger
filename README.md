<h1>jlogger</h1>
<p> JLogger is a library for print colored log's in java</p>

<h2>How to use?</h2>
<p>
JLogger logger = new JLogger();<br>
// Success log <br>
logger.success("Log");<br>
// Error log<br>
logger.error("Log");<br>
// Message log<br>
logger.message("Log");<br>
// Warning log<br>
logger.warning("Log");<br>
// Log<br>
logger.log("Log");<br>
// Exception log<br>
logger.exception("Log");<br>
</p>

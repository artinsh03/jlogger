<h1>jlogger</h1>
<p> JLogger is a library for print colored log's in java</p>

<h2>How to use?</h2>
<div>
  <h4>Create a JLogger Object</h4>
    JLogger logger = new JLogger();
  <h4>Success log </h4>
    logger.success("Log");
  <h4>Error log</h4>
    logger.error("Log");
  <h4>Message log</h4>
    logger.message("Log");
  <h4>Warning log</h4>
    logger.warning("Log");<br>
  <h4>Log</h4>
    logger.log("Log");<br>
  <h4>Exception log</h4>
    logger.exception("Log");<br>
</div>
